## 🔄 Easy Customization

### **Switch Seasons**
Edit `config/site_config.json`:
```json
{
  "seasonalTheme": "christmas"  // or "default"
}
```

### **Update Jobber URL**
```json
{
  "jobberUrl": "https://your-jobber-link"
}
```

### **Change Phone**
```json
{
  "phoneNumber": "(917) 397-0128",
  "phoneTel": "tel:19173970128"
}
```

### **Swap Background**
```json
{
  "themes": {
    "christmas": {
      "backgroundImage": "https://your-image-url"
    }
  }
}
```
